### Prerequisites

**Cargo & Solana**

### Cloning the repository

```shell
git clone https://github.com/leopico/Solana-Bootcamp-Counter.git
```

### Install packages

```shell
cargo check
```

### Start for testing

```shell
cargo test -- --nocapture
```
